hp_airport.ipynb - тетрадка с кодом, где реализуются взятые нами алгоритмы (Louvain algorithm, Leiden algorithm) и метрики (closeness centrality, eigenvector centrality)

- airport_indexes.csv -> файл с результат выделения сообществ с ID аэропортов + номером сообщества
- airport_centrality.csv -> файл с ID аэропортов и его метрикой closeness centrality
- hp_results.csv -> файл с результатами для фанфиков по Гарри Поттеру, структура: имя, размер вершины по собственному вектору
